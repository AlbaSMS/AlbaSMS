Glossary

HTML
A markup language used to structure web pages.
CSS
A style sheet language used to control the look and layout of web pages.
JavaScript
A scripting language that makes web pages interactive, such as handling button clicks or animations.
Elements
The basic building blocks of HTML that define content and structure.
Attributes
Extra information inside an element's opening tag that modifies its behavior.
Table
Used to present tabular (rows and columns of) data. Tables should NOT be used to provide layout.
Query string syntax
The part of a URL after a ?, containing key-value pairs (key=value) separated by & to pass data to the server.
Controls Support
The assistance and mechanisms provided to ensure that system controls (like policies, procedures, or tools) function properly and consistently.
